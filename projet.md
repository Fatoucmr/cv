Sujet du projet :
Recueil de recettes en Markdown

Description du projet : 
Ce projet consiste à créer un recueil de desserts inspirées de séries et de film entièrement en Markdown.  
Chaque recette contiendra le nom, les ingrédients, les étapes de préparation, ainsi qu'une image, des vidéos ou des GIFs. 
Le recueil sera ensuite converti en PDF ou en page HTML grâce à Quarto.

Je vais donc utilisé :
- Markdown
- Quarto pour la conversion en PDF/HTML
- Visual Studio Code
